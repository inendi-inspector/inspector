Coding rules
============

You must be *strict* with those points

* Documentation managed by Doxygen (http://www.doxygen.net)
* Testing done with CTest
* Core functions operated with libapr (http://apr.apache.org)
* Python can control the library: 
  - must provide a high level bindings for the GUI
* Windows (=Visual Studio) and Linux (=gcc) portable
* Linux kernel coding style
